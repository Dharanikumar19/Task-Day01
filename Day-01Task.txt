1.Difference between HTTP1.1 vs HTTP2
Answer: 
i) HTTP2 is musch faster and more reliable than HTTP1.1
ii) HTTP1.1 loads a single request for every TCP connection,
    while HTTP2 avoids network delay by using multiplexing.
iii) HTTP is a network delay sensitive protocol in the sense that
     if there is less network delay,then the page loads faster.
iv) As opposed to HTTP1.1, which keeps all requests and responses in plain text format,
    HTTP2 uses the binary framing layer to encapsulate all messages in binary format.

2.Objects and its internal representation in Javascript
Answer:
i) Objects in javaScript is the most important datatype 
  and forms the building blocks for modern JavaScript.
ii) An object, is a reference data type.
iii) Objects are more complex and each object may contain any combination of 
   these primitive data-types as well as reference data-types.
iv) Variables that are assigned a reference value are given a reference 
    or a pointer to that value.
v) Objects in JavaScript may be defined as an unordered collection of related data, of
   primitive or reference types, in the form of key-value pairs.
